#### 3.3.1 v8 → v9 모델 수정 사유

v8 모델은 "냥수디는 늑대채린이 직접 영입한 인물 → 미러링이 보호로 반전" 이라고 진술했음. README v9 정정으로 **유치원 컨텐츠가 크루 단위 협업** 임이 밝혀짐 → "늑대채린 직접 영입" 이라는 v8 의 객관적 attribution 은 부정확.

그러나 본 계정의 실제 발화는 변하지 않음:
- `늑아가 순혈직계 냥수디` (214459)
- `늑아가의 후예답게` (214154 코멘트)
- `순혈 냥수디팬` (214154 코멘트)

즉 **객관 사실 ≠ 본 계정의 인지**. 본 계정은 객관적으로 크루 영입인 인물을 **주관적으로 늑대채린 직계로 호명** 함으로써 보호 카테고리에 편입시킴. 이는 미러링 모델의 결정 변수가 **(a) 영입의 객관적 사실** 이 아니라 **(b) 본 계정의 호명·카테고리화 행위** 라는 것을 시사.

#### 3.3.2 v9 수정 모델

```
미러링 신호 = if 본_계정_호명 in {"늑아가 직계", "순혈", "후예", "유스" 등 직계 호명}:
                "보호" (적대 신호 반전)
              else if any(외모, 실력, 포지션, 팬이동):
                "적대"
              else:
                "중립/친근"
```

핵심: **호명 행위 자체가 카테고리 결정의 마지막 게이트**. 객관적 인덕션 경로는 호명 가능성을 높이는 **사전 조건** 일 뿐, 결정 변수는 호명 그 자체.
