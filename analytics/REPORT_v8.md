#### 3.3.2 예외 조건이 왜 필요한가

냥수디는 v7 4축 모델에서 **외모 ✓ + 팬이동 ✓** 로 두 축이 동시에 작동 → 다축 OR 모델은 적대 예측. 실측은 7건 전수 우호. 단일 모델로 설명 불가.

해결: README v8 가 **"늑대채린의 '유치원' 컨텐츠를 통해 합류"** 를 명시하고 있음. 본 계정의 자기 발화 (§3.12) 가 냥수디를 **"늑아가 순혈직계"·"늑아가의 후예"** 로 반복 프레이밍하고 있음. 즉 냥수디는 본 계정 의식 속에서 **늑대채린의 확장된 자아(extended self)** 로 분류되어 있어, 미러링의 **타겟이 아니라 공유 대상** 이 됨.

일반화: **미러링 = "우상을 위협하는 타자" 에 대한 적대** 인데, "우상이 데려온 사람" 은 타자가 아니라 **우상의 일부**. 따라서 적대 축이 켜져도 신호가 반전.
