README v7가 "또해영은 늘씬한 몸매와 예쁜 외모, 온화한 성격으로 1년간 케이대에서 활동하며 많은 팬을 만들었다. 늑대채린 본인이 또해영 탈퇴이전부터 자주 견제성 발언을 행한 적 있음" 을 명시 — 미러링 가설의 가장 깨끗한 자연 실험.

- **또해영의 프로필 = 견제 코드 3축(외모·성격·매력·신규 매력) 모두 충족**.

→ 우상의 견제 + 팬의 미러링 + 자리 승계 서사가 한 인물에서 모두 관측. 단순 "팬덤 적대" 모델로는 또해영을 단독 설명 불가.

1. **공격 대상이 거의 전부 늑대채린의 잠재 견제 대상** (남성 멤버 김학수·뽀뇨·정민기·고석현·홍덕·장윤철·정영재 등은 대부분 거의 거론 안 됨).

**가설 B — 디씨 더케이갤러리 익명 김학수 비난자.** v5 보강 그대로. `scrape_dc_thek` 후 즉시 검증.

**가설 C — 늑대채린 견제 코드 미러링 (v6 재구성).** 단순 팬심 → **우상의 견제 코드 내면화**. §3.3 의 9개 케이스 일대일 대응 + 또해영 결정적 케이스(§3.3.1a) 가 핵심 증거. 결정적 검증: 늑대채린 본인의 방송 발화 (특정 멤버에 대한 견제성 발언) 와 본 계정의 게시 시점 간 시간 상관 분석. 24~72시간 lag 가설 검증 가능 (가설 H).

**가설 D — 증폭자 vs 원인 제공자.** v5 증폭자 결론 유지. v6 추가: 코드 클러스터 전체의 합산 영향력은 단독 증폭자보다 무거움.

**가설 E — 경상도 집단 타겟팅.** 직접 지역 혐오 0건. 구조적 교차 사실만.

**가설 F — 정치 동기 부수성.** §7 결론: 양적 1.2% 부수적이지만 **방향은 일관되게 반-보수**. 외국 공작원 동조 직접 증거 없음.

**가설 G — fandom 위치.** v5 G1·G2·G3 병기. **v6**: G3 "분란 조장자" 의 발생 메커니즘이 **fandom의 자기-선택**으로 설명됨 (§3.3.3). 즉 G3는 본 계정만의 기질이 아니라 늑대채린 fandom의 구조가 만들어내는 위치.

**가설 H (v6 신설) — 견제 발화의 시간 상관.** 늑대채린의 방송 견제 발화 → 본 계정의 미러링 게시 사이의 시간 lag 가 **24~72시간** 일 것이라는 예측. 검증 방법: 늑대채린 방송 다시보기에서 견제성 발언 시점 라벨링 + 본 계정 동일 인물 거론 글의 시점 분포 비교. 만약 lag 이 평균 36시간 이내·표준편차 24시간 이내로 좁게 모이면 가설 C 의 강한 입증.

**가설 I (publish v1 신설) — fandom 정치 동질성.** §7.5 — 견제 코드 보유 클러스터가 정치 성향에서도 동질일 가능성. 검증: 호응자 10명 정치 글 패턴 비교.
